3CX Blacklist Management

Web-basierte Verwaltung der 3CX Rufnummern-Blacklist. Einfach. Sicher. Effizient.

Jetzt starten Auf GitHub ansehen

Features

📞

Blacklist-Verwaltung

Rufnummern hinzufuegen, bearbeiten und entfernen. Anrufe und SMS separat blockieren.

🔗

3CX API-Integration

Direkte Anbindung an die 3CX xAPI mit OAuth 2.0 Authentifizierung.

🔍

Echtzeit-Suche

Schnelles Filtern und Durchsuchen aller blockierten Nummern und Beschreibungen.

🔒

Sicherheit

API-Zugangsdaten werden serverseitig gespeichert und nie an den Browser uebermittelt.

Installation

1

Repository klonen

git clone https://github.com/Datenwunder089/3cx-blacklist-management.git
cd 3cx-blacklist-management
2

Abhaengigkeiten installieren

npm install
3

Starten

npm start

Die App ist dann unter http://localhost:3000 erreichbar.
